Config key: `menu`

Types: (s) - string, (b) - boolean, (i) - integer

---

- `items`: ([s]: []) array of object for customi`ing the menu items
  - `name`: (s: '') name of menu item
  - `hide`: (b: false) controls whether the menu item is hidden or not

```json
"onboarding": {
  "items": [],
}
```
